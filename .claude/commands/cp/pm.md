# /cp:pm — Plugin Manager

Claude Plugins 마켓플레이스의 모든 작업을 처리합니다.
**무엇이든 자연어로 말씀해 주세요.**

## 사용 예시

```
/cp:pm 날씨 플러그인 만들어줘
/cp:pm infographic-generator에 스킬 하나 더 추가해줘
/cp:pm infographic-generator에 MCP 서버 추가해줘
/cp:pm 모든 플러그인 검토해줘
/cp:pm 플러그인 목록 보여줘
```

> 새 플러그인 생성은 `/cp:new-plugin`, 품질 검사는 `/cp:validate` 를 직접 사용해도 됩니다.

## 처리 과정

1. **요청 분석** — 무엇을 원하는지 파악
2. **계획 수립** — 필요한 단계와 담당 에이전트 결정
3. **실행** — 단계별 진행 (각 단계에서 확인)
4. **결과 보고** — 완료 내용과 다음 단계 안내
